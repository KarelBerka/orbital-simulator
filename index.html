Matematický popis

Vlnová funkce $\psi_{n,l,m}$ popisuje stacionární stav elektronu. Výskyt elektronu (hustota pravděpodobnosti) je dán kvadrátem vlnové funkce $|\psi|^2$.

$$\psi_{1,0,0}(r) = \frac{1}{\sqrt{\pi}} e^{-r}$$
  • Radiální uzlové plochy ($n-l-1$): 0
  • Úhlové uzlové plochy ($l$): 0
  • Celkem uzlových ploch ($n-1$): 0
